# Installation

This folder hosts **installations**
as spatial and experiential projections.

Installations emphasize:
- embodiment
- spatial relation
- silence and absence
- non-linear access

Typical strengths:
- physical presence
- experiential immediacy
- suspension of narrative

Typical blind spots:
- continuity
- responsibility tracing
- explicit development

Structural constraint:
Installations must remain legible
as projections of structure,
not as immersive totalities.
