# Film

This folder hosts **film** as a synthetic projection
of the life architecture.

Film emphasizes:
- simultaneity
- environment
- contingency
- relational dynamics

Typical strengths:
- situational density
- contextual realism
- non-verbal relation

Typical blind spots:
- responsibility attribution
- structural abstraction
- sustained ambiguity

Structural constraint:
Film carries a high risk
of false coherence.

Any film project must explicitly state
how narrative closure is resisted.
