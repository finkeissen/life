# Essay

This folder hosts **essays**
as analytical projections of the architecture.

Essays emphasize:
- conceptual clarity
- explicit structure
- traceable argumentation

Typical strengths:
- precision
- distinction
- reflective distance

Typical blind spots:
- lived immediacy
- temporal pressure
- embodied conflict

Structural constraint:
Essays must not function
as authoritative explanations.

They are projections,
not keys.
