## Protagonist Set-Up: Plot as Emergent Consequence

The external story should not be “constructed” as plot.

It should **emerge** from a single structural set-up:

> Place the main character so that their mere existence generates conflict.

This requires a protagonist who:

- carries **order** internally
- treats **responsibility as non-delegable**
- refuses **violence** and enforcement
- refuses **moral superiority**
- does not act to “win” or “fix” anything

Under these conditions, the system reacts.

Not because the protagonist attacks,
but because **order without power** exposes instability
and becomes a provocation in chaotic dynamics.

### Emergent escalation pattern

1. He sees clearly.
2. He does not force.
3. Others act from understandable motives.
4. Local compensation amplifies global instability.
5. His non-participation is read as threat.
6. He becomes a projection surface.
7. The system stabilizes against him.

No villain is required.
No heroic victory is permitted.
No catharsis is expected.

The plot becomes a consequence of structure.

### The Sisyphean dimension (without instruction)

The work does not recommend attitudes.

But if responsibility returns and cannot be outsourced,
then “carrying” becomes the only remaining mode.

Not as virtue.
Not as growth narrative.
Not for gratitude.

Simply because there is no external instance left.
