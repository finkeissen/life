# Admissibility & STOP Conditions

Before adding any element, test:

1. Does it remove protection?
2. Does it increase irreversibility?
3. Does it expose non-delegability?
4. Does it avoid moral resolution?

If not, it is inadmissible.

STOP conditions:

- If repetition becomes decorative → remove.
- If emotional relief appears → cut.
- If explanation replaces embodiment → rewrite.
- If monumentality becomes motivation → reduce.

Expansion must be structurally justified.
