# Structural Invariants

The work must not contain:

- Solutions
- Redemption
- Catharsis
- Progress narrative
- Therapeutic framing
- Didactic explanation
- Moral superiority
- Heroic resolution

Height ≠ solution  
Turning point ≠ improvement  
Ending ≠ closure  

Open ending is mandatory.

No character is fully guilty.
No character is fully innocent.

Failure is systemic, not malicious.
