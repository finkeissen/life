# Compression Principle

Expansion must be structurally justified.

Rules:

1. Every repetition must alter meaning.
2. Every scene must remove protection.
3. Every passage must increase irreversibility or exposure.
4. No explanation without structural consequence.
5. Emotional relief must not stabilize the system.
6. Redundancy equals deletion.

If a section can be removed without altering structural exposure,
it must be removed.

Compression is not aesthetic minimalism.
It is structural discipline.
