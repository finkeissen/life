# Architecture – Core Principles

This folder contains the non-negotiable structural rules of the project.

The architecture defines:

- Ontological logic of the world
- Structural invariants
- Zone mechanics (A–E)
- Recurrence logic
- Responsibility mechanics
- Admissibility constraints
- STOP conditions

Nothing in this folder is ornamental.
Everything here constrains execution.

If a narrative decision conflicts with this folder,
the narrative decision is removed.
