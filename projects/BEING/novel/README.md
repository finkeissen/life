# Novel System

This directory contains the complete structural, conceptual, and operational architecture of the novel system.

The project is not primarily organized narratively, but structurally.

---

## Structure Overview

### /architecture
Formal system architecture.

Contains:
- structural invariants
- ontology
- zone model
- energy and compression principles
- recurrence logic
- architecture decision log

Recommended reading order:
00 → 12

---

### /ingredients
Content building blocks.

Contains:
- anthropological conditions
- institutional fields
- life phases (structural)
- physical dimension
- structural tensions
- scene and topic ideas


This is the material pool.

---

### /dramaturgy
Macro- and micro-dramaturgy.
(Plot architecture, tension progression, structural movement.)

Currently under development.

---

### /execution
Concrete implementation layer.
(Scenes, chapters, versioning, text.)

Currently under development.

---

## Workflow Logic

1. Define architecture.
2. Structure ingredients.
3. Derive dramaturgy.
4. Write execution.

No isolated scene production without structural embedding.
