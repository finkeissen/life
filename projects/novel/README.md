# Novel

This folder hosts the **novel** as one projection
of the life architecture.

The novel functions as an **interior projection**.

It emphasizes:
- temporal extension
- self-observation
- unreife as lived experience
- gradual shifts in self-understanding
- narrative as subjective coherence

Typical strengths:
- access to consciousness
- development without external resolution
- subtle misalignment over time

Typical blind spots:
- bodily immediacy
- simultaneity
- collective dynamics

Structural constraint:
The novel must not function as
an explanatory master narrative.

It is a projection,
not an interpretation of the whole.
